### 一个例子解决函数参数设置问题
**需求：写一个学生注册的函数**  
1. 位置参数
```python
def login(name,gender):
    print('name:',name)
    print('gender:',gender）
```
特点：姓名和性别是必须的，少了要跳错，多了也不行，name和gender就是位置参数。  

2. 默认参数  
追加需求：输入学生户籍，基本都是杭州本地的学生，个别外地学生。  
```python
def login(name,gender,city='hz'):
    print('name:',name)
    print('gender',gender)
    print('city:',city)
```
